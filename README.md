# chess_ai_engine
CS AI project,  using alpha-beta searching with additional heuristics and pruning. implemented the basic interface for the UCI protocol

# Techni
- alpha-beta pruning
- move ordering
- Transposition Table
- Null-move pruning
- Multi-cut pruning
- Futility Pruning
- Late Move Reduction
